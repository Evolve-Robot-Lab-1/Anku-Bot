### 2. Camera Configuration Fixed
**Problem:** Camera not detected on new RPi

**Solution:** Updated `/boot/firmware/config.txt`:
```
# Changed:
dtoverlay=vc4-fkms-v3d  →  dtoverlay=vc4-kms-v3d  (Full KMS for libcamera)
start_x=1              →  #start_x=1              (Disabled legacy camera)
dtoverlay=ov5647       →  #dtoverlay=ov5647       (Auto-detect handles it)

# Kept:
camera_auto_detect=1
```

**Camera format fix:**
- Default format was `nv21` (YUV) - not compatible with web_video_server
- Changed to `RGB888` which outputs `bgr8` - works with web_video_server

**Working camera command:**
```bash
ros2 run camera_ros camera_node --ros-args -p format:=RGB888 -p width:=640 -p height:=480
```
