## Troubleshooting Notes

### Camera Not Detected
1. Check physical CSI cable connection
2. Verify `camera_auto_detect=1` in `/boot/firmware/config.txt`
3. Reboot RPi after config changes
4. Check with `v4l2-ctl --list-devices`

### Video Not Showing in Web Panel
1. Hard refresh browser: `Ctrl+Shift+R`
2. Test direct stream: http://192.168.1.16:8080/stream?topic=/camera/image_raw
3. Restart robot service if needed
4. Check `/tmp/video.log` for errors
