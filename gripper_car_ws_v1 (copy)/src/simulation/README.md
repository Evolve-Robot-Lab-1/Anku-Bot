Simulation packages skeleton

This folder contains minimal skeleton packages for simulation and testing:

- `arm_sim` - arm-focused simulation and tests
- `base_sim` - mobile-base-focused simulation and tests
- `sensors_sim` - isolated sensor simulation tests
- `mobile_manipulator_sim` - full-system integration simulation

These are intentionally minimal; populate `launch/` and `config/` with actual Gazebo/ros2_control configurations when ready.
